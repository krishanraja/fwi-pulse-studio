## 5. Pricing Psychology

### Anchoring
- Always show Enterprise pricing first
- Pro feels affordable by comparison
- Free establishes floor value

### Decoy Effect
- Pro monthly ($99) vs annual ($948 → $79/mo)
- Savings messaging: "Save $240/year"

### Loss Aversion
- "You're missing 67% of market signals on Free"
- Downgrade friction (confirm data loss)

### Social Proof
- "Join 500+ fractional executives"
- Customer logos on pricing page
- Real-time signup notifications

## 8. Competitive Moat

### Defensibility Factors

1. **Data Network Effects**
   - More users → more signal sources
   - User behavior as implicit signal
   - Community-generated insights

2. **Methodology IP**
   - Patentable composite calculation
   - Published research credibility
   - Academic partnerships

3. **First-Mover Advantage**
   - Brand association with "fractional index"
   - Media relationships
   - SEO authority

4. **Switching Costs**
   - Historical data lock-in
   - Integration dependencies
   - Workflow habits

### Competitive Responses

| Competitor Move | Our Response |
|-----------------|--------------|
| Large player enters | Double down on niche expertise |
| Price undercut | Emphasize quality, methodology |
| Data source overlap | Exclusive partnerships |
| Feature copy | Accelerate innovation |
